# ماكرو سرعة - SR3H MACRO

## نظرة عامة
**ماكرو سرعة، يعطيك السرعة.**

تطبيق ماكرو احترافي مخصص للاعبين يوفر أدوات متقدمة لتنفيذ ضغطات مخصصة للماوس ولوحة المفاتيح مع دقة عالية في التوقيت.

**الموقع الرسمي:** www.SR3H.com

## المتطلبات
- Windows 10 أو أحدث (64-bit)
- .NET 6.0 Runtime
- اتصال بالإنترنت للتفعيل الأولي

## الميزات الرئيسية

### A - Key Sequence Trigger
- تنفيذ تسلسل مفاتيح معقد
- دعم مفاتيح التحضير والضغط والتحرير
- تحكم دقيق في التوقيت بالملي ثانية

### B - Keyboard to Mouse Macro
- ربط مفاتيح الكيبورد بإجراءات الماوس
- دعم النقر الأيسر والأيمن
- 4 أوضاع تشغيل مختلفة:
  - Repeat while holding: تكرار أثناء الضغط
  - No Repeat: تنفيذ مرة واحدة
  - Toggle: تشغيل/إيقاف بالضغط
  - Sequence: تنفيذ تسلسل محدد

### C - Mouse Button to Keyboard Macro
- ربط أزرار الماوس بمفاتيح الكيبورد
- دعم جميع أزرار الماوس (G4, G5, إلخ)
- نفس أوضاع التشغيل المتاحة

## إعداد قاعدة البيانات (Supabase)

### 1. إنشاء مشروع Supabase جديد
1. اذهب إلى [supabase.com](https://supabase.com)
2. أنشئ حساب جديد أو سجل الدخول
3. أنشئ مشروع جديد

### 2. تنفيذ SQL Script
1. اذهب إلى SQL Editor في لوحة تحكم Supabase
2. انسخ محتوى ملف `Database/supabase_setup.sql`
3. نفذ الكود لإنشاء الجداول والدوال

### 3. إعدادات الاتصال (محمية ومشفرة)
البيانات محفوظة بشكل آمن ومشفر في التطبيق. لا حاجة لتعديل أي شيء.

### 4. إضافة المستخدمين
```sql
INSERT INTO macro_subscriptions (email, order_id, is_active, subscription_start) VALUES 
('user@example.com', 'SR3H123', true, NOW());
```

## البناء والتشغيل

### متطلبات التطوير
- Visual Studio 2022 أو أحدث
- .NET 6.0 SDK

### خطوات البناء
```bash
# استنساخ المشروع
git clone [repository-url]
cd MACRO_SR3H

# استعادة الحزم
dotnet restore

# بناء المشروع
dotnet build --configuration Release

# تشغيل التطبيق
dotnet run
```

### إنشاء ملف تنفيذي
```bash
# إنشاء ملف تنفيذي مستقل
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```

## نظام التفعيل

### آلية العمل الجديدة (محدثة)
1. المستخدم يدخل البريد الإلكتروني فقط
2. التطبيق يولد Hardware ID تلقائياً (CPU + MAC + HDD)
3. يتم إرسال البيانات إلى جدول `macro_subscriptions`
4. التحقق من البريد والتفعيل (`is_active`)
5. إذا `hardware_id = NULL` → يتم حفظ الـ ID الجديد
6. إذا `hardware_id` مطابق → قبول مع تحديث `last_check`
7. إذا `hardware_id` مختلف → رفض مع خيار إعادة الربط

### إعادة ربط الاشتراك
- يتطلب البريد الإلكتروني ورقم الطلب
- يتم إعادة تعيين Hardware ID للسماح بربط جهاز جديد
- فترة سماح 30 يوم للعمل بدون اتصال

## الاستخدام

### 1. التفعيل الأولي
- شغل التطبيق
- أدخل البريد الإلكتروني المرتبط بالاشتراك
- اضغط "تفعيل الاشتراك"

### 2. إعداد الماكرو
- فعل الأقسام المطلوبة (A, B, C)
- حدد المفاتيح والأزرار
- أضف الإجراءات المطلوبة
- اضبط التوقيت والتأخير

### 3. تشغيل الماكرو
- اضغط "Apply Settings" لحفظ الإعدادات
- اضغط "Start Macro" لبدء التشغيل
- استخدم المفاتيح المحددة لتنفيذ الماكرو

## الأمان والخصوصية
- جميع بيانات التفعيل مشفرة محلياً
- Hardware ID يُولد من مكونات الجهاز الأساسية
- لا يتم جمع أي بيانات شخصية إضافية
- الاتصال بالخادم فقط للتحقق من التفعيل

## استكشاف الأخطاء

### مشاكل التفعيل
- تأكد من صحة البريد الإلكتروني
- تحقق من اتصال الإنترنت

### مشاكل الماكرو
- تأكد من تشغيل التطبيق كمدير
- تحقق من صحة أسماء المفاتيح والأزرار
- تأكد من عدم تعارض المفاتيح مع تطبيقات أخرى

### رسائل الخطأ الشائعة

- "خطأ في توليد معرف الجهاز": مشكلة في قراءة مكونات الجهاز
